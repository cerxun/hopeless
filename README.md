# hopeless
Единственный интерпретатор языка Hope - Hopeless
Для сборки и установки:

**войдите в свою систему под обычным пользователем
**скачайте архив
**раскройте его командой tar -xzf hopeless.tgz
**при этом появится директория hopeless, зайдите в неё
**дайте команду make
**войдите в систему под пользователем root, зайдите в вашу директорию hopeless
**дайте команду make install
**Интерпретатор устанавливается под именем /usr/local/bin/hopeless, библиотечные модули устанавливаются в директорию /usr/local/share/hopeless; страничка man устанавливается в /usr/local/man/man1 и может быть прочитана командой man hopeless.
